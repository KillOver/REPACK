# FrontLab
free4lab new front

## 开发者安装 Developer Install (from git)

Install Node.js

Clone 

    git clone https://github.com/free4lab/FrontLab.git

Switch to develop branch

    git checkout -b develop origin/develop
    
Install Dependency

    npm install
    
Run WebServer

    gulp
    
Release

    gulp dist:build
    
Clean

    gulp clean
