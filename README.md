## 开发者安装 Developer Install (from git)

Install Node.js

Clone 

    git clone https://github.com/zrss/Front.git
    cd Front
    git checkout develop
    
Install Dependency

    npm install
    
Build CSS

    gulp build
    
Run WebServer

    gulp
